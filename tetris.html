Welcome to Tetris!

Instructions:

  1. Use the left and right arrow keys to move the tetrominoes.
  2. Use the down arrow key to make the tetrominoes fall down faster.
  3. Use the up arrow key to rotate the tetromino.
  4. When a row gets completely filled up, it is removed and the score increases by 10.
  5. The game ends when no more tetrominoes can fall down.
